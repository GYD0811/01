欢迎来到"格罗比言·全球语"项目！

格罗比言·全球语 (Globien) 是新一代的人造国际语。

格罗比言传承世界语 (Esperanto) 精神，融合中英法三大语言，更加简单易学，有望成为更有效的国际沟通桥梁。

格罗比言官网 (https://globien.gitee.io) 由本项目支撑。
